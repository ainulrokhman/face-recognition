Getting Started

Welcome to the high-performance Face Recognition DNN API Documentation. This service handles CNN-based face detection (via YuNet) and deep learning face embedding matching (via SFace) using OpenCV's optimized C++ neural network backend.

Base URL: http://localhost:5000
The API runs locally inside a Docker container. CORS is enabled dynamically to accept requests from your CodeIgniter site (e.g. http://localhost:8080).

Register Student Face

POST

Crops the face, aligns it via CNN landmarks, extracts a 128-dim vector embedding, and saves it in SQLite.

POST /api/register
Request Body
Parameter Type Requirement Description
label_id integer REQUIRED Unique numeric ID of the student.
username string REQUIRED Name of the student.
frame string REQUIRED Base64-encoded image data URL.
target_pose string OPTIONAL Pose target for biometric verification. Options: front, left, right, any.
* Defaults to front if omitted.
* Pass any to **bypass pose verification checks entirely** (e.g. for simple single-photo registrations or legacy devices).
* If only front is registered, face recognition will succeed for front-facing angles but may have reduced accuracy at extreme side angles.
Code Examples
curl -X POST http://localhost:5000/api/register \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_API_KEY_HERE" \
  -d '{
    "label_id": 101,
    "username": "Budi",
    "frame": "data:image/jpeg;base64,/9j/4...",
    "target_pose": "front"
  }'

    method: 'POST',

    },
    body: JSON.stringify({
        label_id: 101,

});

POST /api/delete
Request Body

label_id integer REQUIRED The unique label ID to delete.
